# API-activering van berichten in Adobe Journey Optimizer

Leer hoe te om REST APIs voor contextuele, gepersonaliseerde en in real time transactie en marketing mededelingen te gebruiken.

>[!VIDEO](https://video.tv.adobe.com/v/3422169/?learn=on)

Terwijl Adobe Journey Optimizer een moderne, markeringsvriendschappelijke interface en een werkschema verstrekt om gepersonaliseerde, in real time 1 :1 klantenreizen tot stand te brengen en te beheren, moeten de ondernemingen ook tijd-gevoelig, en opdracht kritieke berichten meedelen die op dringende signalen en auteursberichten aan publiek worden gebaseerd zonder reizen te creëren. Deze berichten kunnen wachtwoordvoorinstellingen, eenmalige codes of zelfs updates na de game voor de abonneebasis van sportfans bevatten.

Met API Gedragsde Campagnes, kunnen de ondernemingen een campagne plaatsen en het gebruiken veelvoudige tijden om gepersonaliseerde berichten met echt - tijd contextafhankelijke informatie te leveren door via REST API teweeg te brengen.

In deze Experience League Live-sessie zullen productdeskundigen van Adobe uitleggen hoe u zaken voor transactie- en marketinggebruik kunt uitvoeren met de mogelijkheden voor via API gestuurde berichten in AJO.
