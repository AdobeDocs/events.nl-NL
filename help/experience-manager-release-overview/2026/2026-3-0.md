# 2026.03.0 AEM Release-video

>[!VIDEO](https://video.tv.adobe.com/v/3483060/?learn=on&enablevpops)

De functies in deze release zijn gericht op innovaties in Experience Manager Sites, Forms, Assets en Cloud Foundation, waaronder:

* Sites
   * De Updates &amp; Steun van de Agent van de Productie van de ervaring

* Forms
   * Workflowondersteuning voor IC Editor
   * Ondersteuning voor themaeditor voor kerncomponenten
   * Ondersteuning voor versies voor Forms

* Assets
   * AEM Content Advisor - GA met Adobe-apps
   * Microsoft Office-connector (PowerPoint, Word, Excel)

* Cloud Foundation
   * Governance Agent: Brand Management
   * Controlemachtigingen voor aanraakinterface
   * OIDC-afmeldingsfunctie
   * Onderhoudsupdates pauzeren

* Cloud Manager
   * Cloud Manager MCP Server
